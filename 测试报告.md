## 1. Bug 列表

### 🔴 高危 Bug (P0) - 必须修复

| Bug ID | 标题 | 严重程度 | 模块 | 描述 | 复现步骤 | 建议修复方案 |
|--------|------|----------|------|------|----------|--------------|
| BUG-001 | 管理接口缺乏权限验证 | 🔴 高危 | 后端/API | `/api/cards`、`/api/admin/batch-delete` 等管理接口未验证Token，任何人可调用 | 1. 使用curl不带Token访问 `GET /api/cards`<br>2. 不带Token发送 `DELETE /api/admin/batch-delete` | 添加中间件验证 `Authorization` Header |
| BUG-002 | 登录接口无暴力破解防护 | 🔴 高危 | 后端/登录 | 可无限次尝试密码，无速率限制 | 1. 编写脚本循环发送登录请求<br>2. 观察是否被拦截 | 添加登录失败次数限制（如5次/分钟/IP） |
| BUG-003 | XSS漏洞 - 备注字段未转义 | 🔴 高危 | 前端/管理后台 | 备注字段直接渲染，可执行脚本 | 1. 在备注输入 `<script>alert(1)</script>`<br>2. 查看是否弹窗 | 使用 Vue 的 `{{ }}` 插值或 `v-text` 自动转义 |

| Bug ID | 标题 | 严重程度 | 模块 | 描述 | 复现步骤 | 建议修复方案 |
|--------|------|----------|------|------|----------|--------------|
| BUG-004 | 密码明文存储和传输 | 🟠 中危 | 后端/登录 | 密码 `jc123` 以明文方式校验，存在泄露风险 | 1. 查看后端代码 `adminLogin` 函数<br>2. 发现直接字符串比较 | 使用 bcrypt 等哈希算法存储密码哈希 |

| Bug ID | 标题 | 严重程度 | 模块 | 描述 | 复现步骤 | 建议修复方案 |

| BUG-008 | 备份路径安全检查可绕过 | 🟡 低危 | 后端/备份 | `strings.HasPrefix` 检查可能被绕过 | 1. 构造特殊路径 `backup/../other`<br>2. 测试是否能访问其他目录 | 使用 `filepath.EvalSymlinks` 后再检查 |

| 问题ID | 标题 | 类型 | 描述 | 解决方案 |
|--------|------|------|------|----------|
| BLOCK-001 | SQLite并发写入性能瓶颈 | 性能 | 高并发批量添加时可能出现数据库锁定 | 考虑使用连接池或迁移到PostgreSQL |

## 4. 修复建议优先级

### 第一阶段 (上线前必须完成)

1. **添加管理接口权限验证中间件**
   ```go
   func authMiddleware() gin.HandlerFunc {
       return func(c *gin.Context) {
           token := c.GetHeader("Authorization")
           if !validateToken(token) {
               c.AbortWithStatusJSON(401, Response{Code: -1, Message: "未授权"})
               return
           }
           c.Next()
       }
   }
   ```

2. **添加登录速率限制**
   ```go
   // 使用令牌桶或固定窗口算法
   limiter := rate.NewLimiter(rate.Limit(5), 5) // 每分钟5次
   ```

3. **修复XSS漏洞**
   ```vue
   <!-- 使用Vue自动转义 -->
   <span>{{ card.remark }}</span>
   ```
