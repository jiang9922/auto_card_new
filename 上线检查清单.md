### 7.2 回滚步骤

1. **停止当前服务**
   ```bash
   # Docker
   docker stop autocard
   
   # Railway
   # 在Dashboard中点击 "Redeploy"
   ```

2. **恢复数据库（如需要）**
   ```bash
   # 使用最近备份
   cp ./backups/cards_xxx.db ./cards.db
   ```

3. **回滚到上一版本**
   ```bash
   git checkout HEAD~1
   # 重新构建部署
   ```

4. **验证回滚成功**
   - 服务启动正常
   - 核心功能正常
